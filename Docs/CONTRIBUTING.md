## Contributions

We welcome your contributions to improve `Ollama Completions for SQL Server`! Here’s how you can get involved:
- **Report Bugs**: Submit issues for any bugs or unexpected behavior.
- **Request Features**: Suggest new functionality or enhancements.
- **Contribute Code**: Help by fixing bugs or adding features.
- **Improve Documentation**: Update or expand documentation for better clarity.

### Contribution Guidelines

- Fork the repository and create a feature branch.
- Ensure code changes are accompanied by unit tests.
- Follow the existing coding style and conventions.
- Submit a pull request with a clear description of your changes.
